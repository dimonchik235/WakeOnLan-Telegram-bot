# WakeOnLan-Telegram-bot
Телеграмм бот запущеный на есп32 посылает WakeOnLan пакет на любое устойство в локальной сети. Позваляет включить свой сервер из любой точки мира 

Краткая инструкция по установке:
1.Создай телеграмм бота в @BotFather и вставь api бота в сооцтетвующее поле в начале кода
2.Введи ssid и пароль от своей сети в соотвецвующее поля в коде
3.Получи свой айди у @userinfobot и вставь его в вайтлист в коде
4.Введи айпи, мак адрес сервера в поля в коде
5.Прошей код на микроконтроллер есп32 и наслаждяйся

Код протестирован на esp32c3 supermini
